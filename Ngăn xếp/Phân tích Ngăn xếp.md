#file [[Ngăn xếp]]

| Thao tác                | Time complexity | Space complexity |
| ----------------------- | --------------- | ---------------- |
| **push()**              | $O(1)$          | $O(1)$           |
| **pop()**               | $O(1)$          | $O(1)$           |
| **top()** và **peek()** | $O(1)$          | $O(1)$           |
| **isEmpty()**           | $O(1)$          | $O(1)$           |
| **isFull()**            | $O(1)$          | $O(1)$           |
